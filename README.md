# 3D-Formula-Graph
This program is meant to act as a visualiser of 3d formuli. This code may be utilised in self-projects, and can be modified in any way. This program uses GLFW( https://www.glfw.org/download.html), GLUT(https://freeglut.sourceforge.net/)and C++ Mathematical Expression Toolkit Library(https://www.partow.net/programming/exprtk/index.html#downloads).
